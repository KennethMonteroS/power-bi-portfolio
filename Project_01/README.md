# Project 01. Employee Performance Dashboard

## Project Overview

This dashboard analyzes sales performance across different regions, products, and time periods.

## Objectives
- Diseñar una base de datos relacional mediante la definición de tablas,
relaciones y tipos de datos adecuados que permitan normalizar la información
dispersa sobre las ventas realizadas.
- Desarrollar consultas básicas y avanzadas en SQL que generen información
clave sobre el rendimiento de la empresa, incluyendo análisis de ventas,
rendimiento de las sucursales, desempeño de trabajadores, etc., para
proporcionar una base sólida para la toma de decisiones.

## Tools
- SQL
- Data exploration
- Aggregations
